# Institutional Onboarding

## First session

1. **Welcome** — operational environment overview
2. **Workspace template** — HL Execution Desk (default) or Full Terminal
3. **Exchange connection** — wallet + Hyperliquid agent approval
4. **Execution desk** — ticket, DOM, slippage, alerts
5. **OmniBar** — `/chart` `/depth` `/exec` `/desk` `/expand` `/help`
6. **Expand workspace** — advanced modules when desk outgrows wedge

## Keyboard workflow

- `Ctrl+K` — OmniBar focus
- Experience bar — density, calm mode, motion
- Panel drag handles — institutional grid layout

## Resume onboarding

PRODUCT panel → **ONBOARD** → **RESUME WALKTHROUGH**

Or dismiss permanently via walkthrough **DISMISS**.
